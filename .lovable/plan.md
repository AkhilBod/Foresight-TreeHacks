

# Foresight Smart Home Commerce

A dark-themed marketing site with a functional app dashboard, inspired by the reference design. Gold/amber accent on deep dark backgrounds, clean typography, smooth professional animations, and custom SVG icons throughout.

## Landing Page

### Navigation Bar
- "Foresight" wordmark logo in gold/amber
- Nav links: How It Works, Features, Security, Pricing, Contact
- "Get Started" CTA button with gold fill and arrow

### Hero Section
- Bold headline: "The Future of Shopping"
- Subtitle explaining AI-powered predictive reordering
- Two CTAs: "Get Started" (gold) and "Request a Demo" (outlined)
- Below: a styled mockup of the Foresight app dashboard showing a purchase approval screen with spending controls and item cards

### How It Works Section
- Step-by-step flow with custom SVG icons (camera, AI brain, cart, card)
- Steps: Observe, Analyze, Suggest, Purchase
- Smooth fade-in animations as each step enters the viewport

### Features Section
- Card grid highlighting: AI Vision Detection, Spending Controls, Smart Reordering, Secure Payments
- Each card with a custom icon, title, and short description
- Subtle hover lift animation on cards

### Security & Trust Section
- Emphasis on fraud protection, encrypted payments, user approval controls
- Trust badges and security-focused messaging
- Clean layout with icon + text pairs

### Pricing Section
- 2-3 tier cards (Free trial, Standard, Premium)
- Feature comparison list per tier
- Gold CTA on recommended plan

### FAQ Section
- Accordion-style expandable questions
- Common questions about privacy, spending limits, supported devices

### Footer
- Foresight branding, nav links, social links (custom SVG icons), legal links

## App Dashboard (separate route: /dashboard)

### Sidebar Navigation
- Menu items: Dashboard, Activity, Devices, Settings, Support
- User profile section at bottom

### Main Dashboard View
- Welcome greeting with daily summary
- Recent activity feed showing detected items and auto-orders
- Spending overview chart (weekly/monthly bar chart)
- Quick action cards: Approve pending purchases, Adjust limits

### Purchase Approval Screen
- List of AI-suggested purchases with item image, name, price, confidence level
- Approve / Decline buttons per item
- Batch approve option

### Spending Controls Panel
- Monthly budget slider
- Per-category spending limits
- Toggle for auto-approve under a threshold

## Design System
- Dark background with subtle grid pattern (matching reference)
- Gold/amber (#D4A843-ish) as primary accent
- Clean sans-serif typography
- No emojis, no Lucide icons -- all custom inline SVGs
- Smooth entrance animations (fade-in, scale-in) inspired by premium fintech aesthetics
- Reactbit-style micro-interactions on buttons and cards
